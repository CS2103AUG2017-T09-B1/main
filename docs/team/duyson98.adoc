== Project: Gentleman's Guide
Gentleman's Guide is a desktop address book application modified from the Address Book - Level 4 to store person contacts and reminders. The user interacts with it using a CLI, and it has a GUI created with JavaFX. It is written in Java, and has about 6 kLoC.

*Code contributed*: [https://github.com/CS2103AUG2017-T09-B1/main/blob/master/collated/main/duyson98.md[Functional code]] [https://github.com/CS2103AUG2017-T09-B1/main/blob/master/collated/test/duyson98.md[Test code]]

*Purpose of portfolio*: This portfolio is written to reflect the contributions made by the author to the Gentleman's Guide.
It shows how all the features implemented by the author work.

==== Justification

`tag` and `retrieve` commands help you organize your address book by putting tags to contacts
and quickly get all contacts that have been assigned with a certain tag.

`untag` and `retag` commands allow you to remove the tags that are no longer needed and modify the existing tags
to reflect the changes in your relationship with the contact owners.
